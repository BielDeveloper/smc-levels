❓ Como Usar o SMC Central

🎮 Instalação Automática

O SMC Central agora suporta instalação automática de fases!

📤 Para enviar sua fase:

  1. Crie sua fase no jogo SMC
  2. Exporte como arquivo .smc
  3. Preencha o formulário na aba "Enviar Fase"
  4. Clique em "Publicar Fase Automaticamente"
  5. Sua fase ficará disponível imediatamente na comunidade!

🎮 Para jogar fases da comunidade:

  1. Navegue pela aba "Explorar"
  2. Clique em qualquer fase para instalação automática
  3. O jogo será aberto automaticamente com a fase carregada
  4. Se não funcionar, use o botão "Download Manual"

🔧 Recursos Avançados:

  • One-Click Install: Clique na fase e jogue imediatamente
  • Auto-Download: Downloads automáticos em segundo plano
  • Game Protocol: Integração direta com o jogo SMC
  • Smart Fallback: Download manual se automático falhar
